This project is a simple Task Management REST API built using Python and Flask, designed to simulate a serverless architecture similar to AWS Lambda and API Gateway. It allows users to create, view, and delete tasks through HTTP requests.
The main motivation behind this project is to understand how serverless backends work and to build a scalable API structure without directly deploying to AWS. It solves the problem of managing tasks efficiently while also demonstrating backend development and API design concepts.
Technologies Used:
Python
Flask
REST API
UUID (for unique task IDs)
JSON (data format)
